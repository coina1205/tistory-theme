Favicon Usage Example

The logo.svg file can be used as a favicon by including the following in your HTML head:

<link rel="icon" type="image/svg+xml" href="logo.svg">

For better browser compatibility, you can also provide PNG fallbacks:

<link rel="alternate icon" type="image/png" href="logo.png">
<link rel="apple-touch-icon" href="logo.png">
